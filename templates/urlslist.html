Urls list

/ GET Renders the home page
/login GET, POST Manages user login: renders the login page or handles login form submission
/logout GET Logs out the user and redirects to the home page
/register GET, POST Manages user registration: renders the registration page or handles registration form submission
/apicalls GET Renders api calls page
/urlslist GET Renders this page
/dashboard GET Renders the profile page for logged-in users
/generatetoken GET Generates a fresh JWT access token for the logged-in user (session-based, for convenience)
/upgrade GET Upgrades the logged-in user's account to premium
/downgrade GET Downgrades the logged-in user's account back to free
